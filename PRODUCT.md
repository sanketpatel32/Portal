# Product

## Register

product

## Users
An individual developer managing project tasks and tracking server runtime telemetry in a unified local workspace.

## Product Purpose
A fast, real-time task manager and backend diagnostic console where developers can track resource usage, chat across sessions, and organize task flow without delay.

## Brand Personality
*   Technical
*   Focused
*   Solid Utility

## Anti-references
*   No decorative gradients (avoid multi-color text gradients, background mesh gradients, or glowing neon slop).
*   No generic, low-contrast SaaS templates.
*   No nested cards or redundant layout sections.

## Design Principles
*   **Technical Restraint**: Focus on solid, high-contrast colors, clear borders, and readable monospaced telemetry. Avoid decorative shadows and blurs.
*   **Aesthetic Clarity**: Rely on typography weight and structural spacing to build visual hierarchy instead of color gradients.
*   **High-Fidelity Telemetry**: Diagnostics and metrics must be clean, precise, and fast.

## Accessibility & Inclusion
*   Strict compliance with WCAG AA (≥ 4.5:1 text-to-background contrast ratio) to ensure maximum legibility under different ambient lighting.
*   Graceful layout scaling down to mobile resolutions.
